# OpenSciFlow Community

Governance, community calls, contribution guides, outreach templates, and meeting notes for OpenSciFlow.

## Channels

Start with GitHub Issues and Discussions. Add Discord/Slack/Feishu only when there are enough active contributors to justify moderation.

## Contributor roles

- Curator
- Manifest reviewer
- Workflow reviewer
- Reproducibility reviewer
- Docs contributor
- Maintainer
